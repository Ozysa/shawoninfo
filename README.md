# Shawon Ahmed - Portfolio

A modern, responsive portfolio website built with React and Vite.

## Features

- **Hero Section**: Eye-catching landing page with introduction
- **About Section**: Education and professional background
- **Skills Section**: Showcase of core competencies
- **Projects Section**: Highlighting key projects (Ozysa Ltd, Marketing Academy, Star Face)
- **Contact Section**: Direct contact information and social media links
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and visit `http://localhost:5173`

### Build for Production

To create an optimized production build:

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```
